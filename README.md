# Personal Profile Website 🌐

## About the Project

This is an AI-based Personal Profile Website created to showcase my introduction, skills, and contact details in a clean and professional format. The project was developed using AI website-building tools and deployed online for public access.

## Features

* Responsive and modern design
* Personal introduction section
* Skills showcase
* Contact information section
* User-friendly interface

## Technologies Used

* AI Website Builder
* HTML
* CSS
* GitHub / Vercel Deployment

## Objective

The main objective of this project is to understand how AI tools can simplify website development and help beginners create professional websites without advanced coding knowledge.
